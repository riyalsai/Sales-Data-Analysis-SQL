# Sales Data Analysis using SQL

## 📌 Project Overview
Analyzed a retail sales dataset to identify top-performing 
regions, products, and sales trends using advanced SQL queries.

## 🛠️ Tools Used
- MySQL
- SQL (GROUP BY, JOIN, HAVING, Window Functions, CTEs)

## 📊 Dataset
- Superstore Sales Dataset
- 9,994 records across 4 regions and 3 categories

## 🔍 Key Business Questions Answered
1. Which region has the highest sales and profit?
2. Which product category performs best?
3. Which customers place the most orders?
4. What is the monthly sales trend?

## 💡 Key Findings
- West region has the highest total sales
- Technology category drives maximum revenue
- Q4 shows highest sales every year

## 📁 Files
- `queries.sql` — All SQL queries used in analysis
- `insights.md` — Detailed findings and business insights
